## ðŸ› ï¸ What Type of Language is JavaScript?

| Feature          | Meaning                                            |
|------------------|----------------------------------------------------|
| **High-Level**    | Easy to read/write. Handles complex stuff (memory, etc.). |
| **Interpreted**   | Runs line-by-line. No need to compile like C or Java. |
